# 🚀 AIOSK 공개 API 테스트 가이드

## 📋 구현 완료된 공개 API 엔드포인트

### 1. 카테고리 목록 조회
```bash
# GET /api/public/categories
curl -X GET http://localhost:3000/api/public/categories
```

**예상 응답:**
```json
[
  {
    "categoryId": 1,
    "name": "커피",
    "sortOrder": 1
  },
  {
    "categoryId": 2,
    "name": "음료",
    "sortOrder": 2
  }
]
```

### 2. 메뉴 목록 조회
```bash
# 전체 메뉴 조회 (FOR_SALE 상태만)
curl -X GET http://localhost:3000/api/public/menus

# 특정 카테고리 메뉴 조회
curl -X GET "http://localhost:3000/api/public/menus?categoryId=1"
```

**예상 응답:**
```json
[
  {
    "menuId": 101,
    "name": "아메리카노",
    "description": "진한 에스프레소와 물의 조화",
    "price": 4500,
    "imageUrl": "/images/americano.jpg",
    "status": "FOR_SALE",
    "categoryId": 1
  }
]
```

### 3. 주문 생성
```bash
# POST /api/public/orders
curl -X POST http://localhost:3000/api/public/orders \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      { "menuId": 101, "quantity": 2 },
      { "menuId": 102, "quantity": 1 }
    ]
  }'
```

**예상 응답:**
```json
{
  "orderId": 1,
  "totalPrice": 13500,
  "status": "RECEIVED",
  "createdAt": "2025-06-15T14:30:00Z",
  "items": [
    { "menuName": "아메리카노", "quantity": 2, "price": 9000 },
    { "menuName": "카페라떼", "quantity": 1, "price": 4500 }
  ]
}
```

## 🔧 구현된 기능

### ✅ 완료된 작업
1. **공개 API 컨트롤러 생성**
   - `src/controllers/public/category.controller.js`
   - `src/controllers/public/menu.controller.js`
   - `src/controllers/public/order.controller.js`

2. **공개 API 라우트 생성**
   - `src/routes/public/category.routes.js`
   - `src/routes/public/menu.routes.js`
   - `src/routes/public/order.routes.js`

3. **관리자 라우트 정리**
   - `src/routes/admin/orders.routes.js` 생성
   - 기존 `order.routes.js` 제거하여 구조 정리

4. **서버 라우트 연결**
   - `server.js`에 공개 API 라우트 추가
   - 관리자 API와 공개 API 명확히 구분

### 🔐 보안 및 검증 사항
- **인증 불필요**: 공개 API는 JWT 토큰 없이 접근 가능
- **데이터 검증**: 메뉴 상태(FOR_SALE), 주문 데이터 유효성 검사
- **에러 처리**: 적절한 HTTP 상태 코드와 에러 메시지 반환
- **실시간 알림**: Socket.IO를 통한 주문 생성 시 관리자 알림

### 🎯 API 경로 구조
```
/api/public/*    - 키오스크용 공개 API (인증 불필요)
/api/admin/*     - 관리자용 API (JWT 인증 필요)
/api/categories  - 관리자 카테고리 관리
/api/menus       - 관리자 메뉴 관리
```
